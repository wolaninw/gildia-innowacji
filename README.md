# Gildia Innowacji — Strona internetowa MVP

Program edukacji przedsiębiorczości dla młodych ludzi.

## Struktura

```
gildia-innowacji/
├── index.html          ← Główna strona z nawigacją i wszystkimi zakładkami
├── assets/
│   └── logo.png        ← Logo Gildii Innowacji
├── pages/
│   ├── gmina.html      ← Landing dla gmin i JST
│   └── szkola.html     ← Landing dla szkół
└── README.md
```

## Zakładki strony głównej

- **Aktualności** — strona startowa z newsami
- **Oferta** → Dla szkół / Dla JST / Dla uczelni
- **Katalog Innowacji** — projekty z filtrowaniem
- **Jak dołączyć?** — 6 kroków
- **Jak wspomóc?** — Patronite, sponsoring, wolontariat
- **O nas** → Cele i wizja / Kontakt / Dane teleadresowe

## GitHub Pages

1. Wgraj folder jako repozytorium na GitHub
2. Settings → Pages → Source: `main` / `/ (root)`
3. Strona pod: `https://twojlogin.github.io/gildia-innowacji`
